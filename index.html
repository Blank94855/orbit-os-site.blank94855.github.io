root@orbit:~$ cat v4.0-alpha1_changelog.txt

Complete UI Overhaul: The "Midnight Blue" Theme

A new professional interface inspired by modern code editors. The sleek, dark theme is easy on the eyes and improves readability across the entire OS, from the lock screen to the terminal.

In-Terminal Wikipedia API

The 'wiki' command now fetches and displays article summaries directly inside the terminal. Get information faster without ever leaving the OS environment.

Graphical System Information

The 'fastfetch' command now launches a dedicated, movable window with detailed system specs, including build number, kernel version, and simulated hardware info.

Full Feature Suite & Draggable Windows

A complete set of tools including 'browser', 'tts', and 'translate' are fully integrated. Media players for both music and video now feature a sleek new design and can be dragged around the screen.

Data Persistence & Customization

Your custom username, chosen font, and login history are now saved between sessions using cookies. Use the 'setname', 'fonts', and 'reset' commands to manage your user profile.
