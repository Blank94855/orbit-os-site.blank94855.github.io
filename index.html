OrbitOS Project Log (july 2024 - Sep 2025)

V1.0 (Pre-Alpha): Simple proof-of-concept. (3 basic commands: neofetch, help, clear)

V2.0 FAILURE: Attempted massive feature rollout but architecture remained broken. Code became messy, prioritizing aesthetics (stars, ugly colors) over stability. **Project Abandoned.**

V3.0+ beta: Complete rewrite focusing on clean, modular code. Introduced stable command map, history, and advanced features (browser, calc). **Current stable version.**
